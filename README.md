# Adhira Local Website

A modern, responsive website for Adhira Local - your local solution for modern web experiences.

## About

Adhira Local is a professional website showcasing local services and solutions. The site features a clean, modern design with smooth animations and responsive layout.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Fast Loading**: Optimized for performance
- **Interactive Elements**: Smooth scrolling navigation and hover effects
- **Accessible**: Built with accessibility best practices

## File Structure

- `index.html` - Main website page
- `styles.css` - CSS styling and responsive design
- `script.js` - JavaScript for interactivity and animations
- `README.md` - This documentation file

## Sections

1. **Home/Hero** - Welcome section with call-to-action
2. **About** - Information about Adhira Local
3. **Services** - Overview of services offered
4. **Contact** - Contact information and details

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript
- Modern web standards and best practices

## Getting Started

Simply open `index.html` in a web browser to view the website. No build process or dependencies required.

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
